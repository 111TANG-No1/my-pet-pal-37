# Pet Pal Hub（宠物助手 Demo）

一个移动端优先的宠物管理 + 宠物AI助手 + 附近发现 Demo（V1：本地数据可演示闭环）。

## 在线演示
- Vercel：https://my-pet-pal-37.vercel.app/

## 核心功能

### 🐾 档案
- 宠物档案管理（基础信息、健康记录等）
- 数据本地持久化（刷新不丢）

### 💬 助手（宠物AI助手）
- 自由对话 + 快捷入口：健康分流 / 喂养训练 / 记录回溯
- 支持“当前宠物”作为默认上下文（可不选）
- 支持“@本条宠物”对单条消息临时指定（发送后回到默认上下文）
- 对话消息本地持久化（刷新不丢）

### 🔎 发现
- 二级Tab：宠友 / 地点
- 支持 列表 / 地图 视图切换
- 宠友：筛选 / 点赞 / 互赞 / 打招呼进入聊天线程
- 地点：搜索（名称/类别/标签/地址等）+ 类别chips + 自定义类别（mock数据演示）

### 👤 我的
- 我的宠物
- 宠友记录（点赞过 / 互赞 / 聊天）
- 数据备份导出（JSON）

## 数据与本地存储
- 宠物、记录、地点、点赞/互赞、聊天线程与消息均持久化到本地存储（localStorage/IndexedDB）
- Schema 变更向后兼容：旧数据不清空、不报错

## 技术栈
- Vite + React + TypeScript
- 本地存储（localStorage / IndexedDB）
- Vercel 部署（Production）
